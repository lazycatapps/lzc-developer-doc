# Docker 支持

如果需要在微服上运行 Docker、Docker Compose 或按传统 NAS 方式维护容器环境，请使用 [LightOS](./advanced-lightos.md)。

LightOS 适合长期保存完整 Linux 运行环境。用户可以从应用商店安装 LightOS 入口应用，创建实例后，在实例内按常规 Linux 方式安装和使用 Docker。实例内的软件、配置和 Docker 数据会随 LightOS 实例持久保存。

如果目标是向普通微服用户分发独立应用，仍应优先制作 LPK。Docker in LightOS 更适合个人开发、调试、运维和自用服务场景。
