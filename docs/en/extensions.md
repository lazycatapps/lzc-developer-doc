# Official Extensions
The LCMD MicroServer team will continue to provide useful extensions to facilitate developers in developing applications. Below are some officially provided extensions:

  - [lzc-file-pickers](https://www.npmjs.com/package/@lazycatcloud/lzc-file-pickers): Web file selection middleware, convenient for LCMD applications to directly open LCMD Cloud Drive and perform file open and save operations
  - [lzc-minidb](https://www.npmjs.com/package/@lazycatcloud/minidb): A small database similar to MongoDB type, convenient for developers to directly develop Serverless applications
