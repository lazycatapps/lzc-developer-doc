## glzjin Expert Application Series
- Developer: [glzjin](https://x.com/glzjin), security expert
- [LCMD MicroServer Application Converter](https://github.com/glzjin/lzc-dtl): Tool to convert Docker Compose applications to LCMD MicroServer format, used for quickly creating Docker applications
- [Xunlei](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.xunlei): Xunlei LCMD MicroServer special edition, requires Xunlei membership to use
- [Nut-WebGUI](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.nutwebgui): View UPS status and automatically shut down LCMD MicroServer when power is out and UPS battery is low
- [Cyberchef](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.cyberchef): "Cyber Swiss Army Knife", a versatile web application designed for various data manipulation tasks
- [RevShells](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.revshells): Powerful tool designed for security professionals and enthusiasts, especially providing support in Capture The Flag (CTF) competitions. This tool provides multiple functions to facilitate penetration testing and ethical hacking activities
- [Certimate](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.certimate): Open source SSL certificate management tool that can help you automatically apply, deploy SSL certificates, and automatically renew them when they are about to expire
- [Trilium](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.trilium): A powerful hierarchical note-taking application designed for building and managing large personal knowledge bases
- [SQLi-Labs](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.sqlilabs): SQLI-LABS is a platform designed for learning and testing SQL Injection techniques
- [Web-Chains](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.webchains): The strongest Swiss Army Knife in Java security research
- [SuperShell](https://lazycat.cloud/appstore/%2Fshop%2Fdetail%2Fin.zhaoj.supershell): WEB management platform integrated with reverse_ssh service, one-click deployment using docker (quick build), supports team collaboration for C2 remote control
- Blog: https://www.zhaoj.in
